================================================================================
  SMM Turk Panel - راهنمای نصب سریع در cPanel
================================================================================

کانفیگ پیش‌فرض دیتابیس:
  DB_NAME: smmturk_tork
  DB_USER: smmturk_tork  
  DB_PASS: Pass1234
  (بعداً رمز را در cPanel عوض کن)

--------------------------------------------------------------------------------
مرحله ۱: ایجاد دیتابیس در cPanel
--------------------------------------------------------------------------------
1. cPanel → MySQL® Databases
2. Create New Database: smmturk_tork (یا هر اسمی - سپس در config.php عوض کن)
3. Create User: smmturk_tork با رمز Pass1234 (یا رمز خودت)
4. Add User to Database → All Privileges

--------------------------------------------------------------------------------
مرحله ۲: آپلود و استخراج ZIP
--------------------------------------------------------------------------------
1. File Manager → public_html
2. آپلود smm-turk-cpanel.zip
3. راست‌کلیک → Extract

--------------------------------------------------------------------------------
مرحله ۳: Import دیتابیس
--------------------------------------------------------------------------------
1. phpMyAdmin → دیتابیس smmturk_tork را انتخاب کن
2. Import → انتخاب install-cpanel.sql → Go

--------------------------------------------------------------------------------
مرحله ۴: تنظیم config.php (اگر نام/رمز دیتابیس فرق کرد)
--------------------------------------------------------------------------------
فایل config.php را ویرایش کن و این مقادیر را درست کن:
  DB_NAME, DB_USER, DB_PASS, SITE_URL

--------------------------------------------------------------------------------
مرحله ۵: تنظیم ایمیل (برای تایید ثبت‌نام)
--------------------------------------------------------------------------------
Admin → Settings → Email: آدرس Mail From را بگذار (مثلاً noreply@دامنه-تو)
این آدرس را در cPanel → Email Accounts بساز اگر نداری.

--------------------------------------------------------------------------------
مرحله ۶: ورود ادمین
--------------------------------------------------------------------------------
URL: https://دامنه-تو/login.php
  Email: admin@smm-turk.com
  Password: password
(حتماً بلافاصله پسورد را عوض کن)

--------------------------------------------------------------------------------
مرحله ۷: Sync سرویس‌ها
--------------------------------------------------------------------------------
Admin → Sync Services → Sync Now

--------------------------------------------------------------------------------
اختیاری: Cron Job (هر ۵ دقیقه)
--------------------------------------------------------------------------------
*/5 * * * * php /home/USERNAME/public_html/cron-sync.php
